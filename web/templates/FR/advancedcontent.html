Le mode avancé est comme le mode facile, sauf que vous pouvez également spécifier des options avancées : des expéditeurs autorisés, des mots-clés et un préfixe. Vous pouvez aussi activer le masquage d'adresse de réponse, et éditer vos adresses jetables. Si vous avez spécifié des options avancées, vous verrez cette page par défaut en vous connectant.

Expéditeurs autorisés: Vous pouvez ajouter et supprimer des « expéditeurs autorisés » ici. Un expéditeur autorisé est soit une adresse complète (jojo@exemple.com) ou un domaine (jojo.com). Les expéditeurs autorisés peuvent envoyer des messages à n'importe laquelle de vos adresses jetables, sans faire avancer le compteur de messages.

Ainsi, vous pouvez vous inscrire à une liste de diffusion dont vous vous méfiez en donnant une adresse jetable. Puis, si vous aimez cette liste, vous pouvez revenir ici et ajouter l'adresse de cette liste dans vos expéditeurs autorisés. Si vous spécifiez un domaine dans vos expéditeurs autorisés, toutes les adresses de ce domaine seront autorisées. Si vous spécifiez une adresse complète, seule cette adresse sera autorisée.

Mots-clés: Les mots-clés vous aident à contrôler la création de vos adresses jetable. Si vous activez l'option « protection par mot-clé » et que vous avez entré au moins un mot-clé, le système ne laissera personne créer une adresse jetable sur votre compte sauf si un de vos mots-clés est dans le premier mot de cette adresse. Exemple : supposons que votre nom d'utilisateur est « spamcowboy » et que vos mots-clés sont « corde » et « selle ». Le système autorisera la création des adresses jetables suivantes:

SangleDeSelle.3.spamcowboy@spamgourmet.com
longuecorde.2.spamcowboy@spamgourmet.com
remonteenselle.6.spamcowboy@spamgourmet.com

mais pas de celles-ci:

coyote.20.spamcowboy@spamgourmet.com
pollueurspammeur.*.spamcowboy@spamgourmet.com

Masquer l'adresse de réponse: Quand le masquage de l'adresse de réponse est activé, le système modifie chaque message transféré pour que, si vous y répondez, la réponse repasse par spamgourmet au lieu d'aller directement à votre correspondant. Spamgourmet modifiera et réacheminera le message pour que votre correspondant voit un message provenant de l'adresse jetable à laquelle il vous a écrit, et non de votre adresse protégée. Votre adresse protégée reste ainsi dissimulée.

Rechercher jetables: Pour voir ou modifier une adresse jetable, entrez l'adresse (indiquée par son 1er mot) et cliquez sur le bouton « rechercher jetables ». Si vous le cliquez sans rien entrer, vous affichez un grand tableau avec toutes vos adresses jetables.

Pour trier les adresses dans la table, cliquez sur l'entête des colonnes.

Dans ce tableau, vous pouvez cliquer sur une adresse jetable (indiquée par son premier mot) et vous pouvez la modifier pour:
a) changer le nombre de messages restants, et
b) si vous le souhaitez, spécifier un expéditeur exclusif pour cette adresse. Un expédieur exclusif peut vous envoyer des messages à cette adresse sans changer le nombre de messages restants. Voir la FAQ pour plus de détails.

Ou, si vous préférez, vous pouvez obtenir le tableau entier au formant XML en cliquant ici.

Préfixe: Si vous êtes vraiment paranoïaque, vous pouvez indiquer un préfixe qui sera exigé sur chaque nouveau jetable créé par un expéditeur non autorisé. C'est en quelque sorte un mot de passe pour créer de nouvelles adresses jetables que vous pouvez changer de temps en temps.

votre préfixe:

Si vous indiquez un préfixe, le système ne créera pas de nouvelle adresse jetable sans le préfixe. Cela vous protège si vous craignez que quelqu'un invente de nouvelles adresses pour votre utilisateur. Le format d'une adresse jetable avec préfixe est:

préfixe.UnMot.x.utilisateur@spamgourmet.com

Ainsi, si votre préfixe est "cowpoke", vous indiqueriez une adresse jetable ainsi:

cowpoke.spamelope.2.spamcowboy@spamgourmet.com

Et si vous changez plus tard votre préfixe en "coyote", le préfixe "cowpoke" ne marchera plus pour créer de nouvelles adresses. Mais les adresses existantes continueront à fonctionner. Si vous supprimez votre préfixe, des adresses jetables peuvent être créées avec ou sans préfixe.

Nbre de messages par défaut: Quand vous créez une nouvelle adresse jetable sans préciser le nombre, c'est ce "nombre de messages par défault" qui est utilisé pour initialiser le compteur de l'addresse. Par exemple, si votre nombre par défaut vaut 3 et que vous créez l'adresse "salut.spamcowboy@spamgourmet.com" (pas de compteur au milieu), le compteur pour cette adresse sera 3, comme si vous aviez écrit "salut.3.spamcowboy@spamgourmet.com".

Liste des messages mangés: Si cette option est activée, le système gardera trace les trois derniers messages mangés et affichera leur expéditeur, leur destinataire et leur horodate.

Ne pas noter messages mangés pour les adr. cachées: Si cette option est activée, le sytème n'enregistrera pas les messages mangés expédiés à vos adresses cachées. Si vous ne voyez pas l'utilité de ces options, ne les activez pas, car elles engendrent une charge supplémentaire pour le système. C'est aussi pour des raisons de performances que nous ne pouvons garder trace que des trois derniers messages mangés.

Ne pas mettre décompte dans sujet: Si vous n'aimez pas le texte supplémentaire entre parenthèses que spamgourmet ajoute dans vos messages (pour vous informer du nombre de messages restants ou du statut autorisé d'un expéditeur), activez cette option. Au lieu d'être dans le sujet, l'information sera alors insérée dans un entête de vos messages, que vous verrez si votre application de courrier électronique affiche les en-têtes SMTP complets.
