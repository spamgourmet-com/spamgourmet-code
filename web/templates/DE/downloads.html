Hier können Sie den freigegebenen Code von Spamgourmet herunterladen. Um Spamgourmet zu benutzen brauchen Sie das nicht zu tun - Sie können sich auf der Hauptseite einfach dafür anmelden. Falls Sie einen ähnlichen Service selbst im Netz anbieten (hosten) möchten oder einfach mal einen Blick auf den Code werfen möchten, sind Sie hier richtig. Wenn Sie selbst am Projekt mitarbeiten wollen, besuchen Sie die Projektseite auf sourceforge.net.
